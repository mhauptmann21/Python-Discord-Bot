# Python Discord Bot

## Description
I created a simple Discord bot using Python so that server members can access easy commands!

## How to Use
To use the bot, an API token is required from the Discord developer site.

## List of Events
* When the program is run, "We are ready to go in, {bot.user.name}!" will print.
* When a member joins the server, the bot will send "Welcome to the server, {member.name}!".
* If a member sends a message with the words "f**k", "b***h", or "s**t", the bot will delete the message and reply "{member.author.mention}, please watch your language!"
* If a member says "python_bot", the bot will reply with "{message.author.mention}, do you need any help?"

## List of Commands
* /hello - the bot will answer back to the author
* /assign - the bot will assign the author the determined role
* /remove - the bot will remove the assigned role
* /dm (add in a message) - the bot will send this message to the author in a dm format
* /reply - the bot will reply to the author's message
* /poll (add in question) - the bot will create an embedded message with a poll and will add reactions
* /secret - the bot will send a message if the author has the authorized role
